Onlineshop:
Woocommerce & Childtheme

Case Study

Introduction

This Project is an onlineshop for cardboard furniture.
The project involved installing Wordpress, preparing the SQL database and adjusting the client-side building a child theme and using pure css.

Start screen of pappmoebelshop.de

Objective

The aim of the project was to relaunch an existing web shop that was based on Magento 1 and to replace Magento with WordPress and WooCommerce.

Start screen with hover menu

Context

The onlineshop has been the center of my work for the past 10 years. I worked on it as a shop manager, not bein gable to fix too many things in the Magento CMS.
Wordpress was easy to work with and offered all the specific features we needed.
